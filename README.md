# Web-Based URL Shortener System

This project is developed as part of an Industrial Internship program in collaboration with upskill Campus and UniConverge Technologies Pvt Ltd (UCT).

The application is built using **Python, Flask, and SQLite** and provides URL shortening with persistent storage and click analytics.
